# AI Resume Screener

AI-powered resume screening automation built using **n8n**, **LlamaCloud**, **Google Gemini/OpenAI**, and **Gmail**.  
This workflow automatically parses resumes, evaluates job-fit, ranks candidates, generates HR interview questions, and sends personalized emails to applicants.
